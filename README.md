# Stock-Market-Analysis

This project is an interactive stock market analytics dashboard built using R and Shiny.

## Features

- Candlestick stock charts
- Moving average analysis
- RSI indicator
- Volume analysis
- Stock comparison
- Download stock data

## Technologies Used

- R
- Shiny
- Quantmod
- Plotly

## Data Source

Stock data is fetched from Yahoo Finance.

## Run the Project

Install packages:

install.packages(c("shiny","quantmod","plotly","TTR","dplyr"))

Run the app:

shiny::runApp()
